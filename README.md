# 📚 DURU - 고촌고 1학년 8반 플랫폼

고촌고등학교 1학년 8반 학생들을 위한 통합 학급 관리 플랫폼

## ✨ 주요 기능

1. **학급 알림** - 공지사항 게시판
2. **학급 일정** - 공동 캘린더 + D-Day
3. **시간표** - 주간 시간표 + 과목 메모
4. **급식** - 오늘의 급식
5. **자료 보관함** - 학습 자료 공유
6. **Tester** - 기능 테스트 페이지

## 🚀 빠른 시작

```bash
# 1. 클론
git clone https://github.com/doetoeri/Duru.git
cd Duru

# 2. 설치
npm install

# 3. 환경변수 설정
cp .env.example .env.local

# 4. 개발 서버 시작
npm run dev
```

## 🛠️ 기술 스택

- **Next.js 14** (App Router)
- **React 18**
- **Tailwind CSS**
- **Supabase** (DB + Auth)
- **TypeScript**
